# TheLand-Project-Presentation
安卓 IOS 端手游 The Land, 仅作为项目展示. 
项目描述: 该项目的开发过程比较有趣, 它原本由韩国人开发, 后来被日本人买断, 然后又外包给我们二次换皮开发. 由于该项目的原本版本非常古老, 使用的各种插件和 SDK 升级困难, 文档难寻, 架构不严谨, 开发和维护的成本都极高.
